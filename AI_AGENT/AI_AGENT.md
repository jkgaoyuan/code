### AI AGENT 
#### 什么是 ai agent 

agent 需要拥有记忆、工具、计划能力 执行能力，主要用于自主完成任务。 


其定义演进迅速没有统一的定义但是主流定义为能自主完成任务的ai智能体


智能体=llms + 记忆+ 规划+ 工具+环境感知
 

 AI AGENT 是能够感知环境、基于目标进行决策并执行动作的智能化应用



#### 趋势 

llm不仅仅是问答它可以被构建为强大的通用问题解决器。

未来模型走向融合， 同时发展出智能体 

 LLM作为大脑，将任务拆解到  executor  （搜索 模型等） 执行 

 本质：大模型作为大脑发号施令  外延攻击决定能力下线， 推理能力决定上限。
