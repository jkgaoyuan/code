### 数据分析处理模型与框架

pandas是python第三方库，其基于numpy实现。
pandas 包含两种数据类型  
a. 一维数组 series 类似于 list
b. 二维数组 dataframe 

import pandas as pd

pandas支持 csv json excel 格式文件的读取， 产生数据流存储为data frame 二维表，在调用内置函数进行数据分析处理。 

绘图

绘图时数据分析的重要工作， python中常用matplotlib， pandas也内置了函数支持绘图

import matplotilb.pyplot as plt 
x = []
y = []
plt.plot(x,y,label='example') #绘制
plt.show() #显示

plt.xticks(x)          #传入x作为刻度
plt.xticks(range(2,25))  #传入range作为数列
plt.yticks(range(min(y),max(y)+1))  #传入最小最大
plt.title('标题')
plt.xtitle('abc')
plt.ytitle('abc')
